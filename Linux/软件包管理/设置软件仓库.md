## 设置软件仓库
>yum命令  
>`yum repolist                     //列仓库`  
>`yum list                            //列软件`  
>`yum clean all                   //清缓存`  
>`yum -y install 软件名...    //安装软件并且安装依赖`  
>`yum -y remove 软件名    //卸载软件，只删除原软件，不删除依赖`   
>`                                       // -y遇到是或否的时候选yes`
