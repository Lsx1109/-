## 设置软件仓库
>- yum命令  
>`yum repolist                     //列仓库`  
>`yum list                            //列软件`  
>`yum clean all                   //清缓存`  
>`yum -y install 软件名...    //安装软件并且安装依赖`  
>`yum -y remove 软件名    //卸载软件，只删除原软件，不删除依赖`   
>`                                       // -y遇到是或否的时候选yes`   

---

>- rpm命令  
>`rpm -q  软件名            //模糊查找一个软件是否已安装`  
>`rpm -ivh  软件名-版本信息.rpm...    //安装软件不安装依赖，添加ivh选项在安装中，看起来便利一些`  
>`rpm -e  软件名...           //卸载软件`
