## 查看mysql数据库的版本号：
    mysql> select version();
		+-----------+
		| version() |
		+-----------+
		| 5.5.36    |
		+-----------+
	
	查看当前使用的是哪个数据库？
	mysql> select database();
	+-------------+
	| database()  |
	+-------------+
	| bjpowernode |
	+-------------+
	
	mysql> show
    -> databases
    -> ;
	+--------------------+
	| Database           |
	+--------------------+
	| information_schema |
	| bjpowernode        |
	| mysql              |
	| performance_schema |
	| test               |
	+--------------------+

	注意：mysql是不见“;”不执行，“;”表示结束！

	mysql> show
    ->
    ->
    ->
    ->
    ->
    ->
    ->
    ->
    -> \c
	mysql>
	\c用来终止一条命令的输入。
