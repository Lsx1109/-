查询一个字段？
		select 字段名 from 表名;
		其中要注意：
			select和from都是关键字。
			字段名和表名都是标识符。
		
		强调：
			对于SQL语句来说，是通用的，
			所有的SQL语句以“;”结尾。
			另外SQL语句不区分大小写，都行。
		
		查询部门名字？
			mysql> select dname from dept;
			+------------+
			| dname      |
			+------------+
			| ACCOUNTING |
			| RESEARCH   |
			| SALES      |

			4 rows in set (0.00 sec)
